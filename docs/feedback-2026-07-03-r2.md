## R2-G2: 自重の仕様変更 ⬜

- [ ] 自重ボタンは「体重の数値を入れる」のではなく「自重で行った」というフラグを記録する(体重と同重量の加重トレと区別するため)
- [ ] 自重フラグ ON のとき、重量欄は「加重分」を意味する(0 = 純自重、10 = 自重+10kg)。表示は「自重」「自重+10kg」
- [ ] 1RM 換算: 設定に体重が登録されている場合のみ「体重 + 加重」を総負荷として Epley 式で換算し、加重セットと比較可能にする。体重未登録なら 1RM は表示しない(無理に換算しない)
  - 採用理由: 加重チンニング等では「体重+加重」の総重量で 1RM を扱うのが一般的で、既存の加重セットの計算(そのままの重量)とも矛盾しない
